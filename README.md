# CUSTOMER-CHURN-PREDICTION-USING-MACHINE-LEARNING
## DESCRIPTION
A telecommunications company is experiencing a high number of customers discontinuing its services. The project is to develop a machine learning model capable of predicting whether a customer is likely to churn so the company can proactively implement retention strategies.
## Dataset used
The Dataset used in this project is the Telco Customer Churn Dataset
## Technologies used
Python, Jupyter, Pandas, Numpy, Matplotlib, Seaborn, Sklearn
## Describe the technologies used
Python was used as the primary programming language for data preprocessing, model development, evaluation, and visualization.Sklearn was the main machine learning framework used for data processing, model training and model evaluation. Matplotlib was used to create graphs and visual representations of the data. Pandas was used for data manipulation and preprocessing while Numpy was used for numerical computations and array operations.
## Installations
Before running the project, the required libraries were installed
## Possible Usage & Application of the data
The model can be used for Customer Churn Prediction, Customer retention, customer segmentation, revenue protection, marketing campaign optimization and improvement of services.
## Results/Key Findings
The Machine Learning Model successfully identified customers likely to churn, long-term customers exhibited lower churn rates while customers with shorter tenure had a higher likelihood to churn.
